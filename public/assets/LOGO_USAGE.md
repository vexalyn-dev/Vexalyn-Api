# 🎨 Vexalyn API Logo Assets

### 1. **logo.svg** (Main Logo - 200x200)
Full logo dengan V letter, API brackets, dan data flow lines.
**Use for:** Homepage hero, about section, footer

### 2. **logo-simple.svg** (Simple Version - 120x120)
V letter di neobrutalism square dengan "API" text.

Logo + "VEXALYN API" text horizontal.

Minimal V letter untuk browser tab.

### 6. **og-image.svg** (Social Share - 1200x630)
Large preview untuk social media sharing.

## 💻 How to Use:

### In HTML:
```html
<!-- Navbar -->
<img src="/assets/logo-horizontal.svg" alt="Vexalyn API" height="40">

<!-- Homepage Hero -->
<img src="/assets/logo.svg" alt="Vexalyn API" width="200">

<!-- Favicon -->
<link rel="icon" href="/assets/favicon.svg" type="image/svg+xml">

<!-- Open Graph -->
<meta property="og:image" content="https://vexalyn-api.up.railway.app/assets/og-image.svg">
```

### In CSS:
```css
.logo {
  background-image: url('/assets/logo-icon.svg');
  background-size: contain;
  background-repeat: no-repeat;
}
```

API Documentation

Complete reference untuk Vexalyn REST API. Semua endpoints, parameter, dan response format yang kamu butuhkan untuk integrate dengan Anichin donghua data.

Anichin Endpoints

Scraping donghua data dari Anichin streaming platform

Aktif
GET
/anichin/home

Home Feed API

Dapatkan data donghua terbaru dari homepage Anichin termasuk featured, trending, dan recommendations.

Cache: 5m Speed: ~3-5s
Aktif
GET
/anichin/search?q=query

Search Endpoint

Search donghua berdasarkan keyword atau judul. Parameter wajib: q (minimum 1 karakter).

Cache: 3m Dynamic
Aktif
GET
/anichin/genres

Genres Endpoint

Daftar lengkap semua genre donghua yang tersedia di Anichin. Jarang berubah, cache lama.

Cache: 1h Static
Aktif
GET
/anichin/ongoing-series

Ongoing Series

Daftar donghua yang sedang ongoing atau sedang tayang. Update berkala sesuai jadwal rilis.

Cache: 5m Live
Aktif
GET
/anichin/popular-series?filter=weekly

Popular Series

Ranking donghua paling populer. Filter: weekly, monthly, atau all.

Cache: 10m 3 filters
Aktif
GET
/anichin/new-movie

New Movie Endpoint

Film donghua terbaru dan dirilis baru dari Anichin NEW MOVIE section.

FAQ

Pertanyaan yang Sering Ditanyakan

Temukan jawaban untuk pertanyaan umum tentang Vexalyn API.

Apakah Vexalyn API gratis untuk digunakan?

Ya, Vexalyn API 100% gratis tanpa batasan rate limit. Kamu bisa menggunakan semua endpoint tanpa perlu API key atau sistem pembayaran.

Apakah perlu API key untuk mengakses endpoint?

Tidak perlu! Semua endpoint Vexalyn API dapat diakses langsung tanpa autentikasi atau API key. Cukup panggil endpoint-nya dan kamu akan langsung mendapat response.

Berapa response time rata-rata API ini?

Response time bervariasi per endpoint: Home Feed (~3s), Search (~3s), Genres (~3s), Ongoing Series (~4s). Dengan sistem caching, request berikutnya bisa di bawah 100ms.

Apakah ada rate limit atau quota harian?

Ada! Saat ini berlaku rate limit 50 request per jam per IP address untuk memastikan layanan tetap stabil dan fair untuk semua developer. Saat limit tercapai, API akan mengembalikan HTTP 429 beserta header X-RateLimit-Remaining dan Retry-After yang menunjukkan kapan limit reset. Limit reset otomatis setiap 1 jam.

Bagaimana cara melaporkan bug atau request fitur baru?

Temukan bug atau punya ide fitur baru? Langsung kunjungi halaman Report Center kami. Isi form laporan dengan detail kategori, deskripsi, dan tingkat keparahan — tim Vexalyn akan menanganinya secepatnya!

Format data apa yang digunakan dalam response?

Semua response dalam format JSON yang clean dan mudah diparsing. Setiap endpoint memiliki struktur data yang konsisten dan terdokumentasi lengkap di Swagger UI kami.
